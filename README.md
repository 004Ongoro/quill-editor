A lightweight, cross-platform code editor built with Electron. Featuring syntax highlighting, debugging tools, and a modern interface.

![QuillEditor Screenshot](assets/screenshot.png)

## Features

- **Lightweight**: Fast and responsive even with large files
- **Syntax Highlighting**: Support for JavaScript, HTML, CSS, Python, and more
- **File Explorer**: Tree view with file/folder operations
- **Tabbed Interface**: Multiple files open at once
