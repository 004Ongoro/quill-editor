# Changelog

All notable changes to QuillEditor will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - YYYY-MM-DD

### Added
- Initial release of QuillEditor
- Lightweight code editor with syntax highlighting
- File tree explorer
- Tabbed interface
- Built-in debug console
- Cross-platform support (Windows, macOS, Linux)
- Context menus
- Status bar with file information
- Activity bar with panel switching
- Tree view with file/folder operations
- Breakpoint debugging support

### Features
- Syntax highlighting for JavaScript, HTML, CSS, Python, and more
- File operations (new, open, save, save as)
- Workspace management
- Console output panel
- Search functionality
- Settings panel
- Keyboard shortcuts
- Responsive UI design
